# Recipy
**Recipe Finder** is a web application designed to help users discover recipes quickly and easily by entering the ingredients they have on hand. The app simplifies meal planning, reduces food waste, and inspires users to try new dishes.

---

## Features

* **Search by Ingredients:** Enter one or multiple ingredients to find recipes that match.
* **Recipe Details:** View detailed instructions, cooking time, and nutritional information.
* **Images:** Each recipe includes a high-quality image for easy identification.
* **Multiple Recipes:** Provides several options for the ingredients entered.
* **Responsive Design:** Works perfectly on desktop, tablet, and mobile devices.
* **API Integration:** Fetches recipes dynamically from public APIs like Spoonacular or TheMealDB.

---

## Technologies Used

* **Frontend:** HTML5, CSS3,  JavaScript
